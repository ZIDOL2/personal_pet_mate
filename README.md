# personal_pet_mate
KH정보교육원 세미프로젝트  
🐾 개발목표  
- 여러가지 반려동물 제품을 비교하여 쉽고 만족스러운 구매 서비스를 제공하는 사이트
- 화면구현부터 백엔드까지 전반적인 구현 경험
- 게시판 CRUD 및 JAVA를 활용하여 다양한 페이지 구현
- AJAX 비동기 처리방식으로 페이지 새로고침 없이 데이터 갱신    

[ZIDOL2 작업내용]
- 로그인, 회원가입, 아이디/비밀번호 찾기
- 공지사항, 자유게시판 CRUD
